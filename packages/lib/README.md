# Код бібліотеки diui

В цій теці знаходиться весь код бібліотеки. Цей пакет не має можливості запускати його в демо-режимі. Для того, щоб подивитись демо, ознайомтесь з іншим пакетом, в якому живе документація: [docs](../docs/README.md).

## NPM Скрипти

Не забутьте встановити все залежності до того, як  будете їх запускати. Щоб в встановити залежності, треба запустити команду `npm install` в **корені всього проєкту, а не тут**.

- `npm run build`: запускає збірку проєкту, яка генерує як не мініфіковану так і мініфіковані версії стилей.
